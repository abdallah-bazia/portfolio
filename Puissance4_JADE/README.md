# Puissance 4 - JADE Multi-Agent System

This is a **Connect Four (Puissance 4)** game implemented with **JADE (Java Agent DEvelopment Framework)** and a Java Swing GUI.  
Two agents play against each other and communicate using JADE ACL messages.

---

## Project Overview
- **Agents:** `PlayerAgent` (one for each player)  
- **GUI:** `Puissance4GUI` (Swing)  
- **Launcher:** `Puissance4Main` (starts the JADE main container and agents)  
- **Purpose:** Demonstrate a simple multi-agent game, agent-to-agent communication, and GUI interaction.
