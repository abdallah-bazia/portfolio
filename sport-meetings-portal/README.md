# Sport Meetings Portal

## Summary
This project is a **web application for connecting sports enthusiasts** in the same city. Users can register, specify the sports they practice, their level, and search for other participants based on sport, level, and city. It was developed as a **university project** to practice **PHP, MySQL, and web development concepts**.

---

## Features
- User registration with personal information (name, surname, city, email).  
- Add sports practiced with level (Beginner, Confirmed, Pro, Supporter).  
- Login system with email and password verification.  
- Search for other users based on sport, level, and city.  
- Dynamic database queries with **JOIN** to combine user and sport data.

---

## Technologies Used
- **PHP**: Server-side scripting for form processing and database interaction.  
- **MySQL**: Database to store users, sports, and their levels.  
- **HTML/CSS**: Front-end forms and styling.
